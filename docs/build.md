# Build {#build}

## Build the projects

### Get Help menu

```
./build.sh

```

### Build and Compile the project

- Build and compile in debug mode

```
./build.sh debug

```

- Build and compile in release mode

```
./build.sh release 

```

### Build and Compile the project and generate documentation

- Build and compile in debug mode

```
./build.sh debug -d 

```

- Build and compile in release mode

```
./build.sh release -d

```

### Build and Compile the project and run static analysis

- Build and compile in debug mode

```
./build.sh debug -a 

```

- Build and compile in release mode

```
./build.sh release -a

```

### Build and Compile All 

- Build and compile in debug mode

```
./build.sh debug -a -d

```

- Build and compile in release mode

```
./build.sh release -a -d

```
