# Tools 
This folder contains the tools necessary to generate reports based on the metrics
gathered during the execution of the tests.

## Valgrind report maker
*remark:*
- **The valgrind report maker has been generated by a LLM.**
- It is a P.O.C
- It contains embedded css and html code.
- The code is blotted and difficult to read & maintain.

### Description
Valgrind is executed and output a memcheck report in a xml file.
The Valgrind report maker reads a xml file and generates an html report.
It generates a report based on the memory usage of all the different tests.
